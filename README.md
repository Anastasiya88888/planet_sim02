# Інструкція до користування програмою

## Додаток дозволяє:
- Моделювання руху планет за законами Ньютона
- 3 готові системи (Сонячна, Подвійна зірка, Три тіла)
- Додавання власних планет
- Зміна швидкості симуляції (1x - 10x)
- Експорт звіту у .txt файл
- Візуалізація орбіт у реальному часі

## Примітка: 
`Цифри вводити на англійській клавіатурі (бо українська 'е' не сприймається програмою)`

## Приклад додавання планети:
- Назва: будь-яка назва (наприклад: `Юпітер`)
- Маса (кг): маса в кілограмах (наприклад: `1.90е27` - це ~1,898 × 10²⁷ кг для Юпітера)
- Відстань (м): відстань від центру в метрах (наприклад: `7.785е11` - це 778,4 млн км)
- Швидкість (м/с): орбітальна швидкість (наприклад: `13070` - це 13,07 км/с)
- Кут: початковий кут від 0 до 360 градусів
